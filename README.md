# E-Commerce Full Stack Project

Stack:
- React + TypeScript for the client
- NestJS + TypeScript for the server
- PostgreSQL for the database

## First Milestone

1. Make sure the client and server run.
2. Connect the server to PostgreSQL.
3. Add the first database entities: users, products, cart items, orders, order items.
4. Build authentication with hashed passwords and JWT.
5. Add products API and product list UI.

## Project Structure

- `client` - React application
- `server` - NestJS API
